# 🚀 Flask Task Manager with Database

## 📌 Overview
This is a Flask-based REST API with SQLite database integration. It allows users to store and manage tasks permanently.

---

## ✨ Features
- Add tasks  
- View tasks  
- Delete tasks  
- Persistent storage using SQLite  

---

## 🛠️ Tech Stack
- Python  
- Flask  
- SQLite
